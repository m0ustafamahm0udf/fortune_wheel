# ربط NodeMCU مع جهازك (Mac/PC) أو Raspberry Pi عبر كابل الـ USB (Serial) لعجلة الحظ

هذا الدليل يشرح لك خطوة بخطوة كيف تقوم بتشغيل النظام المكون من قطعتين:
1. **NodeMCU (ESP8266):** تقوم بإرسال الزاوية باستمرار عبر كابل الـ USB (تمثل حساس الدوران).
2. **جهازك / الراسبيري باي (Python):** سيرفر يستقبل الزاوية عبر منفذ الـ USB، ثم يبثها لاسلكياً عبر شبكة الـ Wi-Fi لتطبيق Flutter (كبديل للـ Raspberry Pi).

---

## 🛑 الخطوة الأولى: تجهيز وبرمجة NodeMCU

1. قم بتوصيل الـ NodeMCU بجهازك عبر كابل الـ USB.
2. افتح برنامج **Arduino IDE**.
3. قم بفتح الملف `NodeMCU_Serial_Sender.ino` الموجود في مجلد `scripts`.
4. تأكد من أنك قمت باختيار البوردة الخاصة بك (`NodeMCU 1.0 (ESP-12E Module)`) من قائمة `Tools` -> `Board`.
5. تأكد من أنك قمت باختيار المنفذ (Port) الصحيح للـ NodeMCU من قائمة `Tools` -> `Port`.
6. قم بالضغط على زر الرفع (**Upload**) لرفع الكود إلى البوردة.
7. بعد الانتهاء، **يجب إغلاق أي نافذة Serial Monitor مفتوحة في الأردوينو** حتى لا تشغل المنفذ أثناء عمل السيرفر في الخظوة القادمة.

---

## 🛑 الخطوة الثانية: تجهيز واكتشاف المنفذ (Port)

لكي يستطيع كود البايثون القراءة من الـ NodeMCU، يجب أن نخبره باسم المنفذ (Port) الذي تم توصيله به.

### لمستخدمي الـ Mac:
1. افتح الـ Terminal.
2. اكتب الأمر التالي لمعرفة المنافذ المتصلة:
   ```bash
   ls /dev/cu.*
   ```
3. ابحث عن نص يشبه `/dev/cu.usbserial-0001` أو `/dev/cu.SLAB_USBtoUART` وقم بنسخه.
4. افتح ملف `scripts/rpi_usb_server.py`.
5. في السطر 11، قم بلصق الاسم الذي نسخته للمتغير `SERIAL_PORT`، هكذا:
   ```python
   SERIAL_PORT = '/dev/cu.usbserial-0001' # ضعه هنا 
   ```

### لمستخدمي Windows:
- في العادة يكون المنفذ بالـ Windows باسم (مثلاً `COM3` أو `COM4`).
- قم بتغيير المتغير لكلمة الـ COM التي وجدتها في Arduino IDE ، هكذا:
   ```python
   SERIAL_PORT = 'COM3'
   ```

---

## 🛑 الخطوة الثالثة: تشغيل السيرفر (كود الـ Python)

الآن وقد برمجنا الـ NodeMCU وحددنا المنفذ، سنقوم بتشغيل السيرفر.

### 1- التثبيت (أول مرة فقط)
في الـ Terminal، تأكد من تثبيت مكتبة `pyserial` عبر هذا الأمر:
```bash
pip3 install pyserial
```

### 2- التشغيل الدائم
لتشغيل السيرفر يجب أن تكون داخل مجلد `scripts`:
1. افتح الـ Terminal وانتقل لمجلد المشروع، ثم إلى مجلد السكربتات:
   ```bash
   cd مسار_المشروع/scripts
   ```
   أو ببساطة من الـ Terminal المدمج في VS Code (تأكد أنك واقف بمسار fortune_eyt/scripts).
2. قم بكتابة أمر التشغيل:
   ```bash
   python3 rpi_usb_server.py
   ```

---

## ✅ الخطوة الرابعة: ربط تطبيق Flutter بالسيرفر
1. بمجرد تشغيل كود البايثون، ستظهر لك رسالة في الـ Terminal تُخبرك بـ الـ IP الخاص بك (مثلاً **192.168.1.6**).
2. كما ستظهر لك رسالة تؤكد الاتصال الـ USB، مثلاً: `[USB] 🔌 Successfully connected to NodeMCU...`
3. افتح تطبيقك، ضع الـ IP في شاشة إعدادات عجلة الحظ (الاتصال بالراسبيري)، واضغط "اتصال".
4. العجلة ستدور مباشرةً ومزامنة مع الزوايا التي يقرؤها السيرفر من سلك الـ USB!

**ملاحظات مهمة:**
- لا تغلق الـ Terminal أثناء التجربة (السيرفر يعمل طالما الـ Terminal مفتوح).
