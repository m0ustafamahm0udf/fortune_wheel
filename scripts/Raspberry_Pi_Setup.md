# 🍓 دليل إعداد وتشغيل الراسبيري باي (Raspberry Pi)

هذا الدليل مخصص لتشغيل السيرفر على جهاز الراسبيري باي باستخدام بيئة عمل افتراضية (Virtual Environment - venv) موجودة على سطح المكتب.

---

## 🛠️ الخطوة 1: إنشاء وتفعيل بيئة العمل (venv)

بما أنك تستخدم venv على سطح المكتب في الراسبيري باي، فإن أول خطوة دائماً هي الدخول إلى هذه البيئة وتفعيلها.

1. افتح نافذة الأوامر (Terminal) في الراسبيري باي.
2. إذا لم تكن قد أنشأت البيئة بعد، استخدم هذا الأمر لإنشائها (باسم `myenv` على سبيل المثال):
   ```bash
   python3 -m venv ~/Desktop/myenv
   ```
3. **لتفعيل البيئة** (يجب فعل ذلك في كل مرة تفتح فيها نافذة جديدة):
   ```bash
   source ~/Desktop/myenv/bin/activate
   ```
   *(ستعرف أنها تفعلت عندما يظهر اسم البيئة `(myenv)` في بداية السطر في الـ Terminal).*

---

## 📦 الخطوة 2: تثبيت المكتبات المطلوبة

بمجرد تفعيل البيئة (venv)، يجب عليك تثبيت المكتبات التي يحتاجها السيرفر للعمل. هذه الخطوة تُنفذ مرة واحدة فقط داخل البيئة.

1. داخل الـ Terminal (والبيئة مفعلة)، اكتب الأمر التالي:
   ```bash
   pip install pyserial websockets
   ```
   *(نستخدم `pip` هنا بدلاً من `pip3` لأننا داخل الـ venv).*

---

## 🔌 الخطوة 3: معرفة اسم منفذ الـ USB (للقطعة الإلكترونية)

نحتاج لمعرفة الاسم الذي يطلقه الراسبيري باي على الـ NodeMCU الموصلة به.

1. قم بتوصيل الـ NodeMCU بالراسبيري باي عبر كابل الـ USB.
2. في الـ Terminal، اكتب الأمر التالي للبحث عن المنافذ:
   ```bash
   ls /dev/ttyUSB*
   ```
   *(في أغلب الأحيان سيكون المنفذ هو `/dev/ttyUSB0`. إذا لم يظهر شيء، جرب `ls /dev/ttyACM*`).*

---

## ⚙️ الخطوة 4: تعديل ملف السيرفر (`rpi_usb_server.py`)

قبل تشغيل السيرفر، تأكد أن المنفذ صحيح.

1. افتح ملف `rpi_usb_server.py`.
2. في الأعلى (حوالي السطر 12)، قم بتغيير السطر الذي يحتوي على `SERIAL_PORT` ليتطابق مع المنفذ الذي وجدته في الخطوة السابقة:
   ```python
   # تغيير هذا السطر ليناسب الراسبيري باي
   SERIAL_PORT = '/dev/ttyUSB0' # غيره إلى اسم المنفذ الذي ظهر لك
   ```
   *(احفظ التغييرات في الملف).*

---

## 🚀 الخطوة 5: تشغيل السيرفر

الآن أصبح كل شيء جاهزاً للتشغيل. تذكر أن هذه الخطوات يجب أن تتم **بينما البيئة الافتراضية مفعلة**.

1. في الـ Terminal، اذهب إلى المجلد الذي يحتوي على السيرفر (تأكد من مسار مجلدك الفعلي):
   ```bash
   cd ~/Desktop/fortune_eyt/scripts
   ```
2. قم بتشغيل السيرفر باستخدام بايثون الموجود في الـ venv:
   ```bash
   python rpi_usb_server.py
   ```
   *(نستخدم `python` وليس `python3` لأن الـ venv يتولى توجيه الأمر للنسخة الصحيحة).*

---

## 🔑 أوامر مختصرة للتشغيل السريع (في المستقبل)

عند فتح الراسبيري باي في المرات القادمة، ستقوم فقط بالتالي:
1. افتح Terminal.
2. فعل الـ venv:
   ```bash
   source ~/Desktop/myenv/bin/activate
   ```
3. اذهب للمجلد وشغل السيرفر:
   ```bash
   cd ~/Desktop/fortune_eyt/scripts
   python rpi_usb_server.py
   ```
---

## 🆘 مشاكل محتملة وحلولها

* **مشكلة `Permission Denied` على منفذ الـ USB:**
  إذا رفض النظام فتح المنفذ، قم بإعطائه الصلاحيات عبر هذا الأمر:
  ```bash
  sudo chmod 666 /dev/ttyUSB0
  ```
  *(غير `/dev/ttyUSB0` للاسم الذي ظهر لك إن كان مختلفاً).*
